# Pygame Project 

## Initial Setup

* Create a folder `pygameproect`
* create sub folders `assets`, `config`, `fonts`, `level`, `sprites`, 'utils`
* create a git ignore file `.gitignore`
* create a readme file for markdown `README.md`
* create a requirements file `requirements.txt`
* create a markdown file to describe the steps 'Setup.md' 

## Installation 
* install python 
* install git
* install Vscode 

## development environment
* set up a virtual environment `py -m venv pygameproject-env`
* activate it `pygameproject-env/Scripts/activate`
* update pip `py -m pip install --upgrade pip`
* install all requirments `pip install -r requirements.txt`

## Admin 
* set up GitHUb account (if you don't have one )
* add files to gitignore (`**/pygameproject-env/*`)
* set up github repo
* run git initialise (`git init`)
